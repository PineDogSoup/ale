# ale

```bash
NAME:
aelfctl - A simple command line go sdk client for AELF

USAGE:
aelfctl [flags]

COMMANDS:
call            Call contract method name with contract method input by string.
contracts       Get the default contract ContractName & ContractVersion & Author & ContractAddress
help            Help about any command
send            Send transaction with contract method input by string.

OPTIONS:
--endpoints=[http://127.0.0.1:8000]       aelf node endpoints
-h, --help[=false]                            help for aelfctl
```
